# Store Migration Tool

A frontend-based migration tool that converts Products, Customers, and Orders CSV/JSON files into a Shopify-compatible JSON structure.

### Tech Stack:
- HTML
- CSS
- JavaScript

### Features:
✔ Upload CSV/JSON  
✔ Convert & structure data  
✔ SEO check (Meta title & description)  
✔ Preview table  
✔ Download migrated JSON  

### How to Run:
Just open `index.html` in your browser.
